# saru-id/agent-plugins

A collection of Claude Code plugins for development tools and language support.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [dart-lsp](./plugins/dart-lsp) | Dart/Flutter language server for advanced code analysis |

## Installation

### 1. Add the marketplace

```
/plugin marketplace add saru-id/agent-plugins
```

### 2. Install plugins

Browse available plugins in the Discover tab:

```
/plugin
```

Or install directly:

```
/plugin install dart-lsp@saru-id-agent-plugins
```

## Requirements

Each plugin may have its own requirements (e.g., language server binaries). See the individual plugin README for details.
